# High Scalability

<aside>
⚠️ Ability to **increase capacity** based on increasing demand of traffic, memory and computing power

</aside>

2 types:

- Vertical Scaling (Scaling Up) - Upgrade to a bigger server
- Horizontal Scaling (Scaling Out) - Add more servers
